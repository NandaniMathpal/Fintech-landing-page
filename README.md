A modern fintech landing page inspired by Razorpay’s design, built with React and Vite. The project demonstrates responsive UI, clean code structure, and creative use of components while maintaining a professional fintech look.


=> Setup Instructions

1. Clone the repository
    git clone https://github.com/NandaniMathpal/Fintech-landing-page.git
    cd Fintech-landing-page

2. Install dependencies
      npm install

3. Run the project
      npm run dev

4. Open http://localhost:5173 in your browser to see the landing page.

=> Frameworks & Libraries Used

React – Frontend library for building components.
Vite – Fast build tool and development server.
CSS / Flexbox / Grid – Styling and layout.
Optional: any additional library you used, e.g., React Router, Framer Motion.

=> Features

Responsive design for desktop, tablet, and mobile.
No-code payment tools section with interactive cards.

=> Evaluation Criteria

Accuracy of UI – Looks similar to Razorpay style.
Responsiveness – Works on all screen sizes.
Code structure & cleanliness – Modular and readable code.
Creativity – Modern design, subtle animations, and visual polish.
